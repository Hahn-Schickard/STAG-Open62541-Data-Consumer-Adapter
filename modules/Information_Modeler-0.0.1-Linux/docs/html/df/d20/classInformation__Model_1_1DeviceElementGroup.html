Detailed Description

This Class Models a List of Elements within the Device.

It may contain more DeviceElementGroup instances, which act as sublists. The implementation of this class exists in Model_Factory::DeviceElementGroupImpl

Attention
This class should be instantiated by Model_Factory::DeviceBuilder only!
