30 
31  Nodes can be excluded from the graph by passing a list of NodeIds as
32  string representation using excludeNodeIds (ex ["i=53", "ns=2;i=453"]).
33 
34  Output is written into filename to be parsed by dot/neato/srfp...
35  """
36  iter = depth

40  else:
41  root = rootNode

44

53  else:
54  for ref in root.references:
