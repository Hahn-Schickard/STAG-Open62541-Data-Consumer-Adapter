242 /* Wikipedia says (https://en.wikipedia.org/wiki/Endianness): Although the
243  * ubiquitous x86 processors of today use little-endian storage for all types of
244  * data (integer, floating point, BCD), there are a number of hardware
245  * architectures where floating-point numbers are represented in big-endian form
246  * while integers are represented in little-endian form. */
