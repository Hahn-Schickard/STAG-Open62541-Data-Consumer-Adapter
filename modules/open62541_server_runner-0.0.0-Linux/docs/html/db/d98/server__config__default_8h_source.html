28 /* Creates a new server config with one endpoint and custom buffer size.
29  *
30  * The config will set the tcp network layer to the given port and adds a single
31  * endpoint with the security policy ``SecurityPolicy#None`` to the server. A
32  * server certificate may be supplied but is optional.
33  * Additionally you can define a custom buffer size for send and receive buffer.
34  *
35  * @param portNumber The port number for the tcp network layer
36  * @param certificate Optional certificate for the server endpoint. Can be
37  * ``NULL``.
38  * @param sendBufferSize The size in bytes for the network send buffer
39  * @param recvBufferSize The size in bytes for the network receive buffer
40  *
41  */
