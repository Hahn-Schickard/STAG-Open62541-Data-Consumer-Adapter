enum HaSLL::SeverityLevelEnum

Seretiy logging levels in hierarchical order.

Heighest order takes presidence. For example, if logging level is set to INFO, DEBUG and TRACE messages will not be show, whereas INFO, WARNNING, ERROR and CRITICAL will be.
