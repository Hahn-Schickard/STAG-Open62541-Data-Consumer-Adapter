◆ ASYNC_LOGGING

#define ASYNC_LOGGING

#include <LoggerRepository.hpp>

Enables asynchrounus logging.

Defaults to OFF

Defined at compile time with -DASYNC_LOGGING or add_compile_definitions(ASYNC_LOGGING) from CMakeLists.txt

◆ CONFIG_PATH

#define CONFIG_PATH   "exampleConfig.json"

#include <LoggerRepository.hpp>

Specifies the path to json configuration file.

This file is read on singelton creation. It set the following options:

  1. Logfile name
  2. Global logging pattern
  3. Global logging level
  4. Duplicate logging to STD stream
  5. Maximum logfile count
  6. Maximum logfile size

◆ MESSAGE_QUEUE_SIZE

#define MESSAGE_QUEUE_SIZE   1024

#include <LoggerRepository.hpp>

Specifies the maximum amount of messages in the queue.

Only used in conjuction to ASYNC_LOGGING.

Defaults to 1024 when ASYNC_LOGGING is ON

Defined at compile time with -DSET_MESSAGE_QUEUE_SIZE="5000" or add_compile_definitions(SET_MESSAGE_QUEUE_SIZE="2048") from CMakeLists.txt

◆ THREAD_COUNT
